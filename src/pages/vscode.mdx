# Data Science Editor for Visual Studio Code

The **Data Science Editor Extension** loads the Data Science Editor in a web view and allows to reference **.CSV files** from the workspace.

![A data analysis program](./hero.png)

## Getting started

-   Download the **[Extension VSIX](https://microsoft.github.io/data-science-editor-excel/vscode/data-science-editor.vsix)**
-   In Visual Studio Code, open the command palette and run **Extensions: Install from VSIX...** and select the downloaded vsix.
