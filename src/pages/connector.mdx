# Connector

Jacdac uses a PCB edge connector with a 3 wire reversible cable.
Technical details can be found in [Jacdac DDK]
(https://github.com/microsoft/jacdac-ddk/tree/main/connector).

![cable plugin into module](./images/cablepluggingintomodule.png)

The connector provides a mechanical click feedback.

![mechanical click for connector](./images/mechanicalclickconnector.png)

The Jacdac cables and connectors are designed to be used as a bus.

![bus cables](./images/bustopology.png)
