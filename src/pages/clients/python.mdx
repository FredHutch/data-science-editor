# pyJacdac - Jacdac for Python

**The python library is still under heavy development**

There are various ways to integrate Jacdac into **desktop python** projects.

-   [GitHub project](https://github.com/microsoft/jacdac-python)

## Pip Install

Add pyJacdac to your project using `pip` (or `pip3` on Windows)

```
pip install pyJacdac
```
