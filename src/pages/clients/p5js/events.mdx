# Events

Similarly to mouse events, you can register handler for Jacdac events through the `jacdac.events` object.
The registeration function are grouped into per service type and keyed with the service camel name (`button`, `potentiometer`, ...).

The readings are grouped per service type and keyed with the service camel name (`button`, `potentiometer`, ...).

```javascript
jacdac.events = {
    "button": {
        "down": (handler: (event: JDEvent) => void) => { ... },
        "up": (handler: (event: JDEvent) => void) => { ... },
    },
    ...
}
```

## Usage

Add the handler registrations in your ``setup`` function.

```p5js
function setup() {
    ...
    jacdac.events.button.down(() => ...)
}
-----
let counter = 0;
function setup() {
  createCanvas(400, 400);
  textAlign(CENTER, CENTER);
  textSize(200);
  fill(255);
  jacdac.events.button.down(() => counter++);
}
function draw() {
  background(0);
  text(counter, 200, 200);
}
```
