# Humidity Extension

The humidity extension reports the relative humidity
in percentage of full water saturation (0.0 to 100.0).
The `humidity` method returns the current humidity.

The program below outputs the the current humidity when it changes by 5 percent:

```blocks
modules.humidity1.onHumidityChangedBy(5, function() {
    basic.showNumber(modules.humidity1.humidity())
})
```
