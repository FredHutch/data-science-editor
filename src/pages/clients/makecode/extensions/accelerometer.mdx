# Accelerometer

This extension supports 3-axis accelerometers. 
The current forces acting on the accelerometer are 
read via the methods 'x', 'y', and 'z', which return
values in earth gravities (g).

A Jacdac accelerometer module translate acceleration values as follows:

| Orientation           	| X value (g) 	| Y value (g) 	| Z value (g) 	|
|-----------------------	|-------------	|-------------	|-------------	|
| Module lying flat     	| 0           	| 0           	| -1          	|
| Module on left edge   	| -1          	| 0           	| 0           	|
| Module on bottom edge 	| 0           	| 1           	| 0           	|

The following program uses the 'x' and 'y' forces to control the position
of an LED on the micro:bit's 5x5 LED matrix:

```blocks
basic.forever(function () {
    const x = modules.accelerometer1.x() * 100
    const y = modules.accelerometer1.y() * 100
    basic.clearScreen()
    plot(x, y)
    basic.pause(100)
})

function plot (x: number, y: number) {
    let dispX = 2
    let dispY = 2
    if (Math.abs(x) > 30) {
        dispX += Math.sign(x)
    }
    if (Math.abs(x) > 60) {
        dispX += Math.sign(x)
    }
    if (Math.abs(y) > 30) {
        dispY += Math.sign(y)
    }
    if (Math.abs(y) > 60) {
        dispY += Math.sign(y)
    }
    led.plot(dispX, dispY)
}
```

## Events

The accelerometer supports a variety of (debounced) events, including:
- *tilt_up, tilt_down, tilt_left, tilt_right*:  emitted when accelerometer is tilted in the given direction.
- *face_up, face_down*: emitted when accelerometer is laying flat in the given direction.
- *freefall*: emitted when total force acting on accelerometer is much less than 1g.
- *shake*: emitted when forces change violently a few times.
- *force_2g, force_3g, force_6g, force_8g*: emitted when force in any direction exceeds the given threshold.

```blocks
modules.accelerometer1.onShake(function () {
    basic.showIcon(IconNames.Heart)
})
```
