# LED Pixel Extension

The LED pixel extension provides APIs for working with
an strip of RGB LEDs, as well as a built-in set of animations.

The number of LEDs in the strip can be figured using the following
method:
```blocks
modules.ledPixel1.configure(10)
```

The brightness of the strip can be set and retrieved as follows:
```blocks
modules.ledPixel1.setBrightness(85)
let brightness = modules.ledPixel1.brightness()
basic.showNumber(brightness)
```

All the LEDs in the strip may be set to a given RGB color
(encoded using )
```blocks
input.onButtonPressed(Button.A, function () {
    modules.ledPixel1.setAll(0xff0000)
    modules.ledPixel1.setPixel(2, 0x00ffff)
})
```
