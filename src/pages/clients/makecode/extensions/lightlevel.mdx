# Light Level Extension

The light level extension reports light level as a floating
point value between 0 and 100.
The `lightLevel` method returns the current light level.

The following program outputs the current light level whenever it changes by 5:

```blocks
modules.lightLevel1.onLightLevelChangedBy(5, () => {
    basic.showNumber(modules.lightLevel1.lightLevel())
})
```
