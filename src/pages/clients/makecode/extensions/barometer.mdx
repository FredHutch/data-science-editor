# Barometer Extension

The barometer extension reports the pressure in hPa 
(the abbreviated name for a hectopascal pressure unit,
which is exactly equal to a millibar).
The `pressure` method returns the current barometric pressure.

The following program outputs the pressure when the barometer
is first connected and then whenever the pressure changes by 0.2 hPa:

```blocks
modules.barometer1.onConnected(function() {
    basic.showNumber(modules.barometer1.pressure())
})

modules.barometer1.onPressureChangedBy(0.2, function() {
    basic.showNumber(modules.barometer1.pressure())
})
```
