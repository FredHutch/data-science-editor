# Thermometer Extension

The thermometer extension reports the temperature in degrees Celsius.
The `temperature` method returns the current temperature.

As is the case with all sensors, an event handler is supplied for when
the sensor value (temperature) changes by a specified amount.

The program below use the two thermometer APIs to print
the current temperature whenever it changes by one degree Celsius:

```blocks
modules.thermometer1.onTemperatureChangedBy(1, function () {
    basic.showNumber(modules.thermometer1.temperature())
})

```
