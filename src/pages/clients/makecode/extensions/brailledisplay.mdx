# Braille Display

A Braille display module "visualizes" braille characters for 

This program implements a counter using button A and display the counter on the Braille display.

```blocks
let count = 0
input.onButtonPressed(Button.A, function() {
    count +=1
    // update display
    modules.brailleDisplay1.showNumber(count)
    basic.showNumber(count)
})
```
